# Проект тестирования API Яндекс.Самокат

## Описание проекта
Проект содержит автоматизированные тесты для API сервиса Яндекс.Самокат.

## Тестируемые ручки API
1. Создание курьера
2. Авторизация курьера
3. Создание заказа
4. Получение списка заказов
## Дополнительная информация для ревьюера
Все тесты проходят успешно, отчет Allure сгенерирован.
